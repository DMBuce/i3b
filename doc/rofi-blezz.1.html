Name

rofi-blezz - Run rofi in blezz mode.

Synopsis

rofi-blezz [menu] [config] [rofi args]

Description

Runs rofi in blezz mode with different starting menu. This is a workaround for rofi’s blezz module not having an equivalent for blezz’s -d option.

The $ROFI_BLEZZ_THEMESTR environment variable will be applied to rofi’s

Authors

rofi-blezz was written by DMBuce.

Distribution

The latest version of rofi-blezz can be downloaded from the i3b project.
