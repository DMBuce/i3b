rofi-blezz(1)
=============

Name
----

rofi-blezz - Run rofi in blezz mode.

Synopsis
--------

'rofi-blezz' [menu] [config] [rofi args]

Description
-----------

Runs rofi in blezz mode with different starting 'menu'.
This is a workaround for rofi's blezz module not having an equivalent for
blezz's `-d` option.

The `$ROFI_BLEZZ_THEMESTR` environment variable will be applied to rofi's
theme. The default is 'entry {placeholder:"Press hotkey";}' .

Authors
-------
rofi-blezz was written by
link:https://github.com/DMBuce[DMBuce].

Distribution
------------
The latest version of rofi-blezz can be downloaded from
link:https://github.com/DMBuce/i3b[the i3b project].
