Usage: pickdate [FORMAT] [DATE]

    Pick a date starting from DATE's month using rofi. FORMAT controls
    the output.

Keybindings:

    Rofi's keybindings for 'kb-custom-1' and 'kb-custom-2' are bound to
    'p' and 'n'. Pressing them goes to the previous and next month,
    respectively.

    By default, movement is controlled by vim-like "hjkl" keybindings,
    Ctrl + f/b/p/n, arrow keys, and Tab.

    Keybindings can be overridden with $PICKDATE_THEMESTR.

Environment Variables:

    PICKDATE_FMT:         The date format to use if no FORMAT is provided.
    PICKDATE_CAL:         The calendar command to use.
    PICKDATE_SPANATTRS:   Pango <span> attributes used on the row of weekdays.
    PICKDATE_THEMESTR:    Theme string used to override config.rasi .

Defaults:

    PICKDATE_FMT='%F'
    PICKDATE_CAL='cal'
    # see https://docs.gtk.org/Pango/pango_markup.html
    PICKDATE_SPANATTRS='bgcolor="#285577" fgcolor="#ffffff" weight="bold"'
    PICKDATE_THEMESTR='configuration {
    	kb-move-char-back: "h,Left,Control+b";
    	kb-move-char-forward: "l,Right,Control+f";
    	kb-row-up: "k,Up,Control+p,ISO_Left_Tab";
    	kb-row-down: "j,Down,Control+n";
    }'
